Проверка таблицы перед запуском

Правила
  • Каждое изображение должно относиться только к одной строке товара.
  • В одной строке допускается до 5 изображений.
  • Изображения не должны выходить за границы своей строки и не должны заходить в соседние строки.
  • В таблице не должно быть объединенных ячеек.
  • Обязателен столбец Наименование — сюда пишем основное название товара для создания кейса.
  • Столбец Доп информация желателен — сюда удобно сводить всю дополнительную информацию по товару: характеристики, состав, размеры, бренд, артикул, примечания.
  • Перед запуском обязательно укажите диапазон строк, которые надо превратить в товары.
